A habit-based, BeReal-style social app where people log streaks publicly (or to a private folder) and dare each other to keep going. The name is a play on "habitual" and "I bet you won't" — the brand leans into that cheeky, dare-energy.

> Currently in development. Built as a Progressive Web App on the T3 stack (Next.js, tRPC, Prisma, NextAuth) with the **Two-Face × Dare Citrus** design system.
