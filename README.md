# Haba-Chewoah

A habit-based, BeReal-style social app where people log streaks publicly (or to a private folder) and cheer each other on. The name is a play on "habitual" — the brand leans into that cheeky, two-face energy.

> Currently in development. Built as a Progressive Web App on the T3 stack (Next.js, tRPC, Prisma, NextAuth) with the **Two-Face × Citrus** design system.
